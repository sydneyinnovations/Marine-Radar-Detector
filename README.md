# Marine-Radar-Detector
The purpose of this project is to build a recreational boating low cost radar detector to assist with collision avoidance

[![IMAGE ALT TEXT HERE](https://img.youtube.com/vi/B-LqXrCJx_g/0.jpg)](https://www.youtube.com/watch?v=B-LqXrCJx_g)

HF Components
-	Ultra Wide Band (UWB) Antenna 3.9GHz To 10.5GHz Printed Circuit Board UWB Antenna
-	Mini-Circuits High Pass Filter SMA, 8500 MHz to 10.5 GHz Passband 50 Ohm, MIN-VHF-7150+
-	Zeenko 100 kHz to 10 GHz Low Noise Amplifier (LNA), ZK09-BM 
-	1 MHz to 10 GHz RF Power Meter Logarithmic Detector, AD8317
